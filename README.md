# order-system-demo
高并发下单系统 / SpringBoot + Redis分布式锁

## 项目名称：高并发商品下单系统
## 技术栈：Spring Boot + MyBatis-Plus + MySQL + Redis + Redisson

### 项目介绍：
独立设计并实现电商高并发下单核心模块，基于 Spring Boot 构建后端服务，完成商品库存查询、订单创建及库存扣减全流程；针对高并发场景下超卖、数据不一致等问题，采用 Redisson 实现分布式锁，解决原生 Redis 锁存在的锁续期、误删、锁失效等问题，保障库存扣减与订单生成的原子性与一致性。

### 项目亮点：
- 引入 Redisson 分布式锁解决高并发下库存超卖问题，保证核心数据准确性；
- 封装全局统一异常处理与结果返回，提升系统健壮性与接口规范性；
- 通过并发压测验证方案有效性，确保库存与订单数据完全一致，无脏数据、无超卖。
